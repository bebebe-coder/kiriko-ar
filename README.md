# キリコAR

スマホでQRコードを読み取ると、ブラウザ上でカメラが開き、光るキリコを表示する簡易WebARです。

## 使い方

1. `index.html` をHTTPSで公開します。
2. `qr.html` を開き、公開URLを入力します。
3. 作成したQRコードを印刷します。

## 次に必要なもの

- **公開先URL**: GitHub Pagesなど、HTTPSで開ける場所
- **印刷用QRコード**: 公開URLが決まってから作成
- **本格3D化する場合**: キリコの `glb` 形式3Dモデル

## 注意

- スマホのカメラ利用には、基本的に **HTTPS公開** が必要です。
- この版は、まず見せるための簡易版です。
- 本格的にする場合は、キリコの **3Dモデル（glb形式）** を作ると、より本物らしく表示できます。
